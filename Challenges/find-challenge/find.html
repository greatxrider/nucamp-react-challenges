Complete each challenge listed below by editing the code inside the JS section of this Codepen. Place your code in the areas indicated by the comments.

Use the Codepen Console (look for the Console button at the bottom left of your screen) to see console.log() messages.

All challenges will use the array named drinkOrders at the top of the JS section.

Challenge 1

  • Console.log() the first object from the drinkOrders array that contains the customerName of 'Juan'. This challenge has been completed for you as an example.

Challenge 2

  • Console.log() the first object from the drinkOrders array that contains the drinkName of 'Mojito'.

Challenge 3

  • Console.log() only the customerName of the object from the drinkOrders array with the id of 3.

Bonus Challenge

  • Use the filter() method to create a new array containing only the objects with the customerName of 'Zachary'. Then, use the find() method on this new array to retrieve and console.log the id of the object with the drinkName of 'Mojito'.
